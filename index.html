###Install Nginx If you are running CentOS 5.x, you can running following commands to install Nginx (Note: you need to login as root first): ``` $ cat </etc/yum.repos.d/nginx.repo [nginx] name=nginx repo baseurl=http://nginx.org/packages/centos/5/x86_64/ gpgcheck=0 enabled=1 EOF ``` ``` $ yum -y install nginx ``` ``` $ /sbin/chkconfig nginx on ``` ``` $ /sbin/service nginx start ``` If you are running CentOS 6.x, you can running following commands to install Nginx: ``` $ cat </etc/yum.repos.d/nginx.repo [nginx] name=nginx repo baseurl=http://nginx.org/packages/centos/6/x86_64/ gpgcheck=0 enabled=1 EOF ``` ``` $ yum -y install nginx ``` ``` $ /sbin/chkconfig nginx on ``` ``` $ /sbin/service nginx start ```

###Install and configure Git The code for the new tech stack are maintained in GitHub private repositories. We want to take advantage of GitHub Wiki and other collaboration tools in addition to Git. First, you need to install Git before you can pull and push code changes to the GitHub. Git 1.7 or above is recommended to access GitHub. On CentOS 5, run the following commands to install latest git (Note: you need to login as root first) ``` $ wget http://pkgs.repoforge.org/rpmforge-release/rpmforge-release-0.5.3-1.el5.rf.x86_64.rpm $ rpm --import http://apt.sw.be/RPM-GPG-KEY.dag.txt $ yum -y localinstall rpmforge-release-0.5.3-1.el5.rf.x86_64.rpm $ yum -y install --enablerepo=rpmforge-extras git ``` On CentOS 6, run the following commands: ``` $ wget http://pkgs.repoforge.org/rpmforge-release/rpmforge-release-0.5.3-1.el6.rf.x86_64.rpm $ rpm --import http://apt.sw.be/RPM-GPG-KEY.dag.txt $ yum -y localinstall rpmforge-release-0.5.3-1.el6.rf.x86_64.rpm $ yum -y install --enablerepo=rpmforge-extras git ``` Check your git version. It should be greater than 1.7.1 (Current version is 1.7.12) ``` $ git --version ``` Assuming that you have registered a GitHub account at http://github.com, here are the steps to set up Git to pull or push code from GitHub. ``` $ git config --global user.name "john" $ git config --global user.email "" ``` Password cache: ``` $ git config --global credential.helper cache $ git config --global credential.helper 'cache --timeout=360000' ```
